# BugBountyToolBox

- `bbp-setup.sh`: A script to setup your bug bounty tool.
    ```sh
    chmod +x bbp-setup.sh
    ./bbp-setup.sh
    ```

- `bbp-recon.sh`: A script to automate the recon process.
    ```sh
    chmod +x bbp-recon.sh
    ./bbp-recon.sh [domain | /path/to/domains.txt | /path/to/asset.txt]
    ```
